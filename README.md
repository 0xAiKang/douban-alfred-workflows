## 豆瓣电影搜索

-------
### 如何使用
1. 安装好 [Alfred](https://www.alfredapp.com/)
2. 下载本插件 [下载链接](https://github.com/0xAiKang/douban-workflows/releases/download/1.0/DoubanMovie.alfredworkflow)
3. 双击安装
4. 快捷键召唤你的 `Alfred` 输入 `dbm {关键字}`
5. 回车即可到达详细百科
